Code Igniter CAS Library
========================

This library is a wrapper for [phpCAS](https://wiki.jasig.org/display/CASC/phpCAS) -- the official [CAS](http://en.wikipedia.org/wiki/Central_Authentication_Service) client from [Jasig](http://www.jasig.org) -- that simplifies configuration in a [CodeIgniter](http://www.codeigniter.com/) app.
